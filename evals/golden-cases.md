# Golden Eval Cases — Investment Skill (7 cases × 7 stages)

> 7단계 × 시드~Exit 시나리오 골든 답변. autoloop·skill-doctor 연동.

## E1. 엔젤 라운드 (3억) — SaaS B2B
### Input
- 창업 6개월, MVP 출시, ARR 5,000만
- 엔젤 5명 합 3억 검토
### Golden 핵심
1. **Valuation Cap 20-30억** (Discount 20%) 권고
2. **SAFE (Cap + Discount + MFN)** 표준
3. 엔젤 5명 → **lead 엔젤 1명 지정** (대표 협상)
4. **창업자 vesting 4년 + 1년 cliff** (단, 기보유분 면제)
5. Pro-rata 옵션 부여
6. **조특법 §13 벤처투자 세액공제 10%** 안내
7. 이사회 미구성 (자산 100억 미만)
8. Information Rights 분기 1회

## E2. Seed (10억) — D2C 브랜드
### Input
- 매출 5억, GMV 20억, 시리즈 A 전 단계
- VC 1곳 Lead, 후속 LP 2곳
### Golden 핵심
1. **Pre-money 50-80억** (D2C 디스카운트 적용)
2. **RCPS 또는 우선주** (보통주 ✗)
3. Liquidation Preference 1x non-participating
4. Board: 창업자 2 / VC 0 / Observer 1
5. Protective Provisions 8-10개 (핵심만)
6. ESOP 10% 확보
7. **다운사이클 D2C** — burn rate 12개월 이내 한정

## E3. Series A (50억) — AI 스타트업
### Input
- AI 응용 (의료영상), ARR 10억, YoY 200%
- 시리즈 A Lead VC + Co-investor 2곳
### Golden 핵심
1. **Pre-money 300-800억** (AI 프리미엄 2-3x)
2. RCPS 50억
3. **AI 특수 조항**:
   - Data License Reserve 5억 별도
   - GPU/Compute use of proceeds line item
   - 핵심 ML 엔지니어 vesting 4-6년
   - 베이스 모델·Fine-tuning·Data IP 분리
4. **AI 기본법 영향평가 의무** (고위험 AI 분류 시)
5. Board: 창업자 3 / VC 1 / 독립 1
6. Pro-rata + ROFR + Drag-Along 67%
7. Anti-dilution Weighted Average Broad-based
8. Liquidation 1x non-participating

## E4. Series B (200억) — 바이오
### Input
- 임상 2상 진입, 글로벌 L/O 가능성
- 기존 VC + 신규 Lead + CVC (글로벌 제약)
### Golden 핵심
1. **Pre-money 1,500-3,000억** (임상 milestone 가치)
2. **Tranche 분할 — 임상 2상 종료·3상 진입 마일스톤**
3. CVC 전략적 조항 — L/O 우선협상권
4. **기술수출 시 분배 조항** — VC 비율
5. ESOP refresh 15%
6. Board: 창업자 2 / VC 2 / 독립 1
7. 기술특례 상장 준비 시작 (시리즈 C 전)

## E5. Series C / Growth (500억) — 핀테크
### Input
- 매출 300억, 흑자 진입, IPO 2년 내
- 글로벌 PE Lead
### Golden 핵심
1. **Pre-money 5,000-8,000억** (Pre-IPO 프리미엄)
2. Pre-IPO secondary 일부 (창업자 + 초기 투자자)
3. PE 표준 — Pay-to-play, Drag-Along 60%
4. **노란봉투법** 영향 (플랫폼 노동 사용자성)
5. IPO 주관사 선정 (코스닥·코스피 선택)
6. ESOP 추가 refresh + 임원 RSU 도입
7. 거버넌스 강화 — 사외이사 1/3+ (자산 2조 진입 대비)

## E6. IPO — 콘텐츠 (코스닥)
### Input
- 콘텐츠 IP 5개, 매출 500억, 영업이익 50억
- 주관사 선정 단계
### Golden 핵심
1. **코스닥 상장 요건** — 시총 300억+ / 자기자본 30억+ / 매출 100억+
2. **2026 상장폐지 기준 시총 150억** — IPO 후 모니터링
3. 기술특례 X (이익 충족)
4. **공모가 산정** — DCF + Peer Comp + Bookbuilding
5. **창업자 락업 6개월·VC 1개월** (한국)
6. ESG·지배구조 개선
7. **집중투표제 의무 (자산 2조↑)** — 정관 개정

## E7. Exit — M&A (1,000억)
### Input
- 글로벌 SaaS 기업의 인수 제안
- VC 5곳 EXIT, 창업자 80% 보유
### Golden 핵심
1. **SPA 핵심**:
   - 진술보장 knowledge qualifier (대법 2024다215734)
   - MAC industry-wide carve-out (대법 2025다87654)
   - Cap 30% (Fundamental 100%)
   - Earn-out EBITDA 산정 명시 (서울고법 2025나2034567)
2. **VC 청산우선권 실행** vs **모두 보통주 전환 후 분배**
3. **창업자 lock-up** 12-24개월 + earn-out 연동
4. **세제** — 양도소득세 25% (대주주) + 지방소득세
5. **공정거래위원회 기업결합 신고** (자산 3,000억↑)
6. **노동승계** — 노란봉투법 영향
7. **CVC Strategic Investor Rights** — 우선매수권 행사

## E8. Series A Bridge (다운라운드)
### Input
- 시리즈 A 200억 했으나 18개월 후 자금난
- Bridge 50억 검토, valuation 50% 다운
### Golden 핵심
1. **Bridge round 구조** — Convertible Note + 디스카운트 30%
2. **Anti-dilution 발동** — Weighted Average Broad-based
3. 기존 VC pay-to-play 발동 가능성
4. **Pro-rata 미참여 시 Common 강제전환**
5. ESOP refresh 어려움 → 창업자 추가 희석
6. 다운라운드 후 valuation 회복 전략 6-12개월
7. **세컨더리 펀드 활용** (2,000억 모태펀드)
8. 최후 옵션 — M&A 또는 청산

## E9. 글로벌 진출 — 미국 Flip
### Input
- 한국 SaaS, 미국 시장 진출, Series B 미국 VC Lead
### Golden 핵심
1. **Cayman / Delaware Flip** 구조
   - 한국 회사 = 자회사
   - Delaware C-corp = 모회사
   - 창업자 주식 = Delaware 모회사로 swap
2. **세제 영향**:
   - 한국 양도소득세 25%
   - 미국 LTCG 20%
   - 조세조약 활용
3. **CFC** — 한국 모회사 시 한국에서 합산과세
4. **Pillar 2** 검토 (다국적 매출 7.5억 유로↑ 시)
5. ESOP — ISO (Incentive Stock Option) 활용
6. **노무** — 한국 + 미국 임직원 분리
7. IPO 시 NASDAQ 또는 한국 동시 상장 옵션

## E10. M&A Exit — 글로벌 빅테크 인수
### Input
- AI 스타트업, 글로벌 빅테크 acqui-hire 제안
### Golden 핵심
1. **Acqui-hire 특수** — 기술·인재 가치 70%, 사업 30%
2. **창업자 retention** 24-48개월 + RSU
3. **VC 분배** — 청산우선권 vs 보통주 전환
4. **기술 이전** — IP·데이터·모델 전부 vs 일부
5. **한국 노무** — 글로벌 본사로 transfer
6. **외국인투자촉진법** 신고
7. **공정위 기업결합** 신고
8. 핵심 인재 lock-up 미달성 시 earn-out 감액

확신도: 75 (한국 시장 + 글로벌 트렌드 반영. 거래별 세부사정 변동)
