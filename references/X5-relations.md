# X5 — 투자자-창업자 관계

## 1. 구조적 긴장

### 이해관계 비대칭

| 차원 | 창업자 | 투자자 (VC) |
|------|--------|-------------|
| 시간 지평 | 장기 (커리어 = 회사) | 펀드 만기 (7~10년) 내 회수 |
| 리스크 | 올인 (1개) | 포트폴리오 (20~30개) |
| 성공 정의 | 지속 가능한 사업 | **파워로:** 10x+ 리턴 필수 |
| 의사결정 | 제품·팀·비전 | 재무 메트릭·엑시트 가능성 |
| 정보 | 완전 (내부) | 불완전 (보드·보고서) |

> **핵심:** VC는 파워로(power-law) 기반 — 소수 대성공이 펀드 전체 수익 결정. **"적당한 인수" vs "거절하고 더 키워라" 갈등의 근원**

### 정보 비대칭

| 방향 | 비대칭 | 완화 |
|------|--------|------|
| 창업자 → 투자자 | 내부 실상 모름 | 정보권·이사회석·보고의무 |
| 투자자 → 창업자 | 펀드 상황·LP 압력·포트폴리오 전략 모름 | 투자자 펀드 만기·후속 펀드 모집 파악 필수 |

## 2. 단계별 관계

### L1-L2 (엔젤~시드): 밀월

- 투자자 수 적음·밀착
- 보드 형식화 낮음 (advisory)
- **갈등:** 지분 비율·초기 조건의 후속 라운드 영향
- **관행:** 엔젤 hands-off. 시드 VC "helpful" 포지션

### L3 (Series A~C): 제도화

- 이사회 석 본격화 (투자자 이사 1~2명)
- 보호조항(veto) 활성화 → [[X2]] |
- **갈등:** 후속 라운드 주도권 / 밸류에이션 눈높이 / 성장 vs 수익성
- **관행:** 분기별 보드 미팅, 월간 investor update

### L4-L5 (Growth~IPO): 긴장 고조

- 독립이사 추가 → 세력 균형
- IPO vs M&A vs 세컨더리 경로 충돌
- **갈등:** 펀드 만기 압력 → 조기 엑시트 / 통제권 vs 거버넌스 / 세컨더리 분배
- **관행:** 독립이사 중재. 차등의결권으로 창업자 보호 → [[X2]]

## 3. 갈등 유형 및 해결

### 주요 갈등

| 유형 | 내용 | 빈도 |
|------|------|------|
| 전략 방향 | 피봇·시장 확장 여부 | 높음 |
| 엑시트 타이밍 | 지금 팔 것 vs 더 키울 것 | 높음 |
| 경영진 교체 | "전문 CEO" 영입 요구 | 중간 |
| 추가 자금 | 후속 투자 거부·조건 | 중간 |
| 보상·지분 | 급여·세컨더리·스톡옵션 | 중간 |
| 정보 공개 | 과도한 정보 요구 vs 선택적 공개 | 낮음 |

### 해결 메커니즘

| 메커니즘 | 설명 |
|----------|------|
| 이사회 거버넌스 | 독립이사 중재, 위원회 (보상·감사) |
| SHA 조항 | 사전 합의된 규칙 (보호조항·drag-along·tag-along) |
| 투자자 교체 | 기존 투자자 지분 세컨더리 매각 |
| 중재·소송 | [KR] 대한상사중재원 / [US] Delaware Chancery Court |
| 대표 해임 | [KR] 이사회 결의 / [US] Board vote |

## 4. 악화 신호

| 신호 | 의미 |
|--------|------|
| 보드 미팅 빈도 ↑ + 의제 사전 공유 강화 | 신뢰 하락 |
| "전문 경영인" 언급 시작 | 대표 교체 검토 |
| 후속 투자 거부 or "어렵다" | 우선순위 하락 |
| 정보 요청 월간 → 주간 빈번화 | 감시 모드 |
| 질문 → 지시로 전환 | 경영 개입 의도 |
| 투자자 간 메시지 상충 | 정렬 붕괴 |

## 6. LP-GP 구조 기초 (BLIND SPOT #10)

| 요소 | 설명 |
|------|------|
| LP | 연기금·재단·패밀리오피스·모태펀드(KR) |
| GP | VC 운영사 — 투자·관리·엑시트 실행 |
| 펀드 만기 | 7~10년 (연장 1~2회 가능) |
| 관리보수 | AUM 2%/년 |
| 성과보수 | 수익의 20% (허들레이트 8% 초과분) |
| **영향** | **GP는 펀드 만기 내 회수 압력 → 후반기 엑시트 push** |

> **창업자가 알아야 할 것:** 투자자의 펀드 연차·후속 펀드 모집이 투자자 행동 예측의 핵심. **펀드 8~9년차 투자자는 어떤 조건이든 엑시트 원함**

## Drag-Along 창업자 보호 협상 가이드

> **Phase 4 보강** — 교차분석 모순 1.4 해소. L6 Drag-Along 기준과 통일.

| 단계 | 투자자 일반 요구 | 창업자 목표 | 협상력 |
|------|-------------------|-----------|----------|
| Seed~A | 50%+1 ~ 66.7% | 66.7%+ (75% 가능) | 낮음 (투자 유치 우선) |
| B~C | 66.7% | 75% + Board Consent | 중간 (트랙션 증명) |
| D+ | 75% | 75% + Board Consent + Min Price Floor | 높음 (Exit 가시성) |

**원칙:** Drag-Along은 낮게 설정하면 후속 라운드에서 올리기 극히 어려움. 초기부터 높은 threshold 확보 필수. 투자 불가 수준 요구는 역효과.

**X4 연결:** Drag-Along 낮음 + 다운라운드 → 소수 투자자(50%+1)만으로 강제 매각 가능 → 예측 실패 증폭 → [[X4-실패패턴]]

## 패턴 라이브러리

### X5-P01: 이해관계 비대칭 — 실패

- **단계:** X5 (횡단)
- **관점:** V1 (전략·실행)
- **작동 조건:** VC는 파워로 기반 포트폴리오 수익(10x+) 필요 / 창업자는 단일 회사 장기 운영 목표 / 펀드 만기(7~10년) vs 커리어 지평 불일치
- **메커니즘:** [펀드 구조 비대칭] → [성공 정의 충돌] → [전략적 대립(조기 엑시트 vs 장기 성장)]
- **신호/지표:** 후속 라운드 방향 불일치, "이제는 그만 팔자"는 투자자 언급, 창업자의 장기 비전과 VC의 단기 회수 목표 명시적 충돌
- **대조 패턴:** → X5-P08 (정기 업데이트로 인한 신뢰 기반)
- **사례:** 본문 §1 이해관계 비대칭 표, 파워로 정의 — "소수 대성공이 펀드 전체 수익 결정"
- **교차참조:** [[L4-그로스]] 펀드 만기 압력, [[X2-거버넌스]] 이사회 갈등 메커니즘

### X5-P02: 정보 비대칭 심화 — 실패

- **단계:** X5 (횡단)
- **관점:** V2 (법률·규제)
- **작동 조건:** 투자자가 내부 실상 불완전 파악 (보드·보고서만 의존) / 창업자가 펀드 상황·LP 압력·포트폴리오 전략 미인지 / 정보권 운영 부실
- **메커니즘:** [정보 공개 불균형] → [투자자 과도 감시 또는 창업자 blind spot] → [신뢰 붕괴]
- **신호/지표:** 정보 요청 월간 → 주간 빈번화, 투자자 질문 → 지시로 전환, 보드 미팅 의제 사전 공유 강화
- **대조 패턴:** → X5-P08 (정기적 정보 공개로 신뢰 유지)
- **사례:** 본문 §2 정보 비대칭 표 — "완화: 정보권·이사회석·보고의무" 기제 없으면 악화
- **교차참조:** [[X2-거버넌스]] 정보권 설계, [[L3-시리즈]] 월간 investor update 관행

### X5-P03: 펀드 만기 압력 → 강제 엑시트 — 실패

- **단계:** X5 (횡단)
- **관점:** V1 (전략·실행)
- **작동 조건:** VC 펀드 8~9년차 진입 / 성과보수(20%) 실현 시간 제한 / 이전 펀드 close 직전 또는 후속 펀드 모집 중
- **메커니즘:** [펀드 만기 임박] → [어떤 조건이든 회수 원함] → [조기 엑시트 강제 또는 품질 낮은 딜 수용]
- **신호/지표:** VC 펀드 연차 진단 (가능한 물어보기), 후속 펀드 모집 공지, "다음 라운드를 준비해야 한다" 투자자 언급 급증
- **대조 패턴:** → X5-P09 (높은 Drag-Along 임계값으로 보호)
- **사례:** 본문 §6 LP-GP 구조 — "펀드 8~9년차 투자자는 어떤 조건이든 엑시트 원함"
- **교차참조:** [[L4-그로스]], [[L5-IPO]] 펀드 만기 압력 신호, [[L6-MA]] 강제 매각 메커니즘

### X5-P04: 보드 미팅 악화 신호 — 실패

- **단계:** X5 (횡단)
- **관점:** V2 (법률·규제)
- **작동 조건:** 보드 미팅 빈도 정상(분기) → 증가(월간 또는 임시 긴급) / 의제 사전 공유 강화 또는 갑작스런 의제 주제 변경
- **메커니즘:** [신뢰 수준 하락] → [감시 강화] → [통제권 분쟁 임박]
- **신호/지표:** 월간 → 주간 미팅 빈도 급증, 보드 자료 요청 상세도 증가, 투자자 질문 답변 제시간 내 요구 강화
- **대조 패턴:** → X5-P08 (정기 일정으로 안정적 거버넌스)
- **사례:** 본문 §4 악화 신호 표 — "보드 미팅 빈도 ↑ + 의제 사전 공유 강화 = 신뢰 하락"
- **교차참조:** [[X2-거버넌스]] 이사회 운영, [[L4-L5]] 단계별 보드 구조 변화

### X5-P05: "전문 경영인" 언급 → 대표 교체 신호 — 실패

- **단계:** X5 (횡단)
- **관점:** V1 (전략·실행)
- **작동 조건:** VC가 "CEO로서 경험이 필요하다" 또는 "전문 경영인이 필요할 때가 되었다"는 언급 시작 / 보드 또는 비공식 대화에서 반복
- **메커니즘:** [경영진 교체 검토 신호] → [창업자 신뢰도 평가 절하] → [대표 해임 또는 공동 CEO 구조 강제]
- **신호/지표:** VC가 유사 업계 "전문 CEO" 네트워크 소개 빈도 증가, 보드 미팅에서 경영 스타일 비판 증가
- **대조 패턴:** → X5-P07 (이사회 과반 유지로 보호)
- **사례:** 본문 §3 갈등 유형 표 — "경영진 교체: 전문 CEO 영입 요구" (빈도 중간)
- **교차참조:** [[L3-시리즈]] 경영진 교체 압박 시기, [[X2-거버넌스]] 이사회 구성과 대표 권한

### X5-P06: 후속 투자 거부 신호 — 실패

- **단계:** X5 (횡단)
- **관점:** V1 (전략·실행)
- **작동 조건:** 기존 투자자가 후속 라운드 참여 거부 또는 "지금은 어렵다" 모호한 답변 / 대기 중인 라운드가 진행 불가 상태 진입
- **메커니즘:** [포트폴리오 우선순위 하락] → [자금 조달 막힘] → [회사 성장 정체 또는 다운라운드]
- **신호/지표:** 기존 투자자 미팅 빈도 감소, "다음에" 또는 "봐야겠다"는 보류 답변 반복, 신규 투자자 소개 중단
- **대조 패턴:** → X5-P08 (정기 업데이트로 신뢰 유지 → 후속 투자 확보)
- **사례:** 본문 §4 악화 신호 표 — "후속 투자 거부 or 어렵다 = 우선순위 하락"
- **교차참조:** [[L3-시리즈]] 후속 라운드 주도권, [[L4-그로스]] 자금 조달 압박

### X5-P07: 이사회 과반 유지 → 창업자 보호 — 성공

- **단계:** X5 (횡단)
- **관점:** V2 (법률·규제)
- **작동 조건:** 투자 시 [창업자 이사] ≥ [투자자 이사 + 독립이사] 구조 확보 / 후속 라운드에서 비율 유지 또는 개선
- **메커니즘:** [이사회 의사결정 통제] → [대표 교체·피봇 강제 방지] → [창업자 전략 자율성 보존]
- **신호/지표:** 이사회 의결 시 창업자 진영 득표율 추적, veto 행사 능력 보유 확인, 독립이사 성향(founder-friendly) 검증
- **대조 패턴:** → X5-P05 (대표 교체 신호)
