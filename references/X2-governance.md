# X2 거버넌스 (Governance)

거버넌스는 투자 단계별 이사회, 투표권, 보호조항, 정보권의 진화를 다루는 횡단축. 엔젤→시드→시리즈→그로스→IPO 단계마다 창업자 주도 → 공동관리 → 조직화된 전문 거버넌스로 변모.

---

## 1. 이사회 구성과 진화

| 단계 | 구성 | 창업자 의석 | 투자자 | 독립이사 | 의사결정 | 주요 특징 |
|------|------|-----------|--------|---------|---------|----------|
| **엔젤** | 3인 | 2명(필수) | 옵저버석[US] | - | 창업자 주도 | 최소 1인[KR], 옵저버는 투표권 없음 |
| **시드** | 3인 | 2명(일반주) | 1석(우선주) | - | 창업자 통제(2/3) | 정보권 동반: 월/분기 재무, Cap Table |
| **Series A~C** | 5인 | 2명(40%) | 2석(40%) | 1명(20%) | 공동관리(다수결) | 월별 상세 보고, 검사권[US] |
| **그로스** | 5~7인 | 1~2명(15~25%) | 2~3석(40%) | 2~3석(35%) | 투자자 우위 | 분기 전략보고, Exit 로드맵 협의 |
| **IPO** | 5~9인 | 1~2명(10~25%) | 감시역 | 50%[KR]/100%[US] | 독립이사 다수결 | 규제 요구: KR 50% 이상, US 100% |

**독립이사 역할[US]**: 이해충돌 중재, 보상·감사위 구성, 재무점검.

**투자자 권리 진화**: 정보권 → 보호조항 → 검사권 → Exit 협의.

## 2. 투표권 구조

### 2-1. 1주 1표 원칙 [KR] [US]
- 보통주(Common): 1주 = 1표
- 우선주(Preferred): 1주 = 1표 (청산 시만 우선)
- **KR 제약**: 상법 제369조 = 강행규정 (차등의결권 금지)
- **실무**: KR 2024년 기준 차등의결권 미도입

### 2-2. 차등의결권(Dual Class Shares) [US]
| 구조 | 투표력 | 장점 | 단점 |
|------|--------|------|------|
| A Class(창업자) | 1주=10표 | 장기 비전 보호, 단기 변동 회피 | 투자자 영향력 약화, 소수주주 우려 |
| B Class(일반) | 1주=1표 | - | - |

Google, Meta, Snapchat, Pinterest 도입. **KR 벤처기업법 2023.11 개정**: 미상장 벤처회사 허용, 조건 = 5년 한정 또는 IPO까지, 스핀오프/M&A 시 자동 소멸.

### 2-3. 우선주 투표권 제약 [US]
표준: 공통주 증자는 우선주 투표 제외. 청산/M&A는 포함.
**협상점 ★**: 창업자 = 투표권 최소화(정보권 유지), 투자자 = 주요 결정만 투표.

## 3. 보호조항(Protective Provisions & Veto Rights)

**정의**: 우선주가 특정 의사결정에 거부권(Veto) 행사. 투자 가치 보호 장치.

### 3-1. NVCA 표준 보호조항

| 항목 | 거부권 주체 |
|------|-----------|
| 청산·M&A | 우선주 다수 |
| 자본구조 변경(증자) | 우선주 다수 |
| 정관 개정 | 우선주 다수 |
| 부채 발행(임계값↑) | 우선주 다수 |
| 배당금(비상황) | 우선주 다수 |
| 이사회 크기 조정 | 우선주 다수 |

### 3-2. 단계별 강도

| 단계 | 기본 항목 | 확장 항목 | 함정 |
|------|----------|---------|------|
| **시드** | 4가지(청산·증자·정관·부채) | - | 투자자 1인 → 거부권 = 전권 |
| **Series A** | 6가지 + 추가 3~4 | 임직원 배임 등 경영 조항 | "과도한 보호" 시작 |
| **Series B+** | 광범위(채용·예산·자산매매) | - | ⚠️ **경영 마비 위험** |

### 3-3. 과잉 보호의 함정 ★

**사례**: Series B에서 월급 5천만원 이상 임원 채용도 투자자 승인 필요 → 우수 인재 영입 6개월 지연.

**문제점**:
1. 경영 속도 저하 (투자자 회의 일정 종속)
2. 성장 기회 상실 (전략 변경 승인 지연)
3. 인재 영입 애로 (채용 거부권)
4. 창업자 동기 저하

**창업자 협상 전략 ★**:
- **Materiality Threshold**: "연 매출 5% 이상"만 거부권 대상
- **Safe Harbor**: 승인된 예산 범위 내 자율
- **기간 제한**: 3년 후 자동 만료
- **CEO Veto**: 창업자도 특정 항목 거부권 보유

## 4. 정보권(Information Rights)

### 4-1. 표준 요구사항 [US]

**월별/분기별**: 손익계산서, 대차대조표, 현금흐름표
**분기/반기별**: 사업보고서, Cap Table, 주요 의사결정 요약
**연간**: 감사 재무제표, 경영계획서, 연간 보고서

### 4-2. 단계별 정보권

| 단계 | 빈도 | 대상 투자자 | 비용 | 비고 |
|------|------|-----------|------|------|
| **엔젤** | 비공식 | 주요 엔젤 | 없음 | 보고 의무 X |
| **시드** | 분기별 | Major(확인필요: $250K~500K) | 무료 | 기본만 |
| **Series A** | 월별+분기 상세 | Major($1M+) | 무료 | 분석 포함 |
| **Series B+** | 월별 | 모든 우선주 | 무료 | 실시간 KPI 대시보드 |

### 4-3. 정보권 vs 검사권(Inspection) [US]

| 구분 | 정보권 | 검사권 |
|------|--------|--------|
| 방식 | 회사 능동 제공 | 투자자 능동 검토 |
| 대상 | 서면 보고서 | 장부·사무실 현장 방문 |
| 비용 | 회사 책임 | 투자자 부담 |
| 제약 | - | 합리적 기간, 기밀 보호 |

**창업자 협상 ★**: 정보권 수용 → 검사권 연 1회 제한 → 기밀 유지 조항 필수.

### 4-4. 정보권 실무 비용

| 항목 | 월별 | 분기별 |
|------|------|--------|
| 재무 준비 | 1~2시간 | 4~8시간 |
| 회계사 검토(선택) | $1~2K | $2~5K |
| 대시보드 자동화 | 0.5시간 | 1시간 |

**확인필요**: 2024~2026년 한국 스타트업 기준.

## 5. 동반매도·강제매도·우선매수

### 5-1. 우선매수권(ROFR)

**정의**: 창업자·공통주가 제3자에게 판매 시, 회사·투자자가 **먼저 같은 조건 내 구매권**.

**메커니즘**: 창업자 판매의사 → 30일 전 고지 → 투자자 30일 내 매수 결정 → 미매수 시 제3자 판매 가능(동일 조건).

**투자자 관점**: 소유권 희석 방지, 부적절한 외부인 차단.
**창업자 관점 ★**: ROFR 자체는 수용 가능 (회사·투자자 보호) → 대가로 Tag-Along 제한 협상.

### 5-2. 동반매도권(Co-Sale / Tag-Along) [US]

**정의**: 투자자가 지분 판매 시, 창업자가 **같은 조건으로 함께 판매 권리**.

**표준**: "우선주 30% 이상 판매 → 공통주 보유자 pro-rata 참여 가능"

**메커니즘**: 투자자 지분 매각의사 → 30일 전 통지 → 창업자 판매 여부 결정 → 투자자+창업자 → 제3자 판매.

**투자자 관점**: 창업자 Lock-Up 완화 (후속 투자자 신뢰).
**창업자 관점 ★**: 투자자 Exit 때 함께 판매 기회 (특히 Secondary 투자자 진입 시 가치) → 협상: 20~30% 이상만 적용.

### 5-3. 강제매도권(Drag-Along) [US]

**정의**: 투자자 50.1~75%가 판매 결정 시, **나머지도 강제 판매**.

**표준**: "우선주 50% 이상 투표로 판매 → 공통주·소수우선주 동일 조건 강제매도"

**메커니즘**: 투자자(50%+) Exit 결정 → 나머지 의견 무시 → 창업자 강제 지분매각.

**투자자 관점**: 100% 일괄 판매 가능 → 거래 상대방 신뢰도 증대.

**창업자 관점 및 함정 ★**:
```
위험: "우리는 $100M에 팔려고 하는데, 
투자자가 거절하면 주식을 들고만 있어야 한다. 
그런데 50% 투자자가 $50M에 팔면 
강제매도로 우리도 $50M에 판다."
```

**창업자 보호 전략 ★**:
1. **Threshold 상향**: 51% → 75% 이상만 강제매도
2. **Board Consent**: 강제매도 전 이사회(창업자 포함) 승인 필수
3. **Minimum Price Floor**: 강제매도 최저가(예: 창립자금의 2배) 설정
4. **Founder Ratchet**: 동반매도 대가로 창업자도 같은 조건 보장

### 5-4. 한국 vs 미국 [KR] [US]

| 항목 | 미국 | 한국 | 비고 |
|------|------|------|------|
| ROFR | 필수 | 선택(보편적) | - |
| Tag-Along | 필수 | 협상 항목 | - |
| Drag-Along | 표준(51~75%) | 드물음(M&A 시 주로) | - |
| 법적 강제성 | Shareholder Agreement | 주주합의서만 | KR 약함 |

## 6. 창업자 보호 장치(Founder-Friendly Terms)

### 6-1. 의장권 유지

**가치**: 회의 주도권 → 의제 설정, 상징적 리더십.
**보험**: CEO ≠ 의장 → CEO 교체 시에도 의장으로 방향 영향, CEO 해임 시 의장 후임 선임 가능.
**협상 조건 ★**: Series A부터 투자자 요구 증가 → 거절 전략: "독립이사 의장 수용 + 창업자 Veto Council 별도 신설"

### 6-2. 슈퍼 의결권(Super Voting Rights) [US] [KR]

**미국**: Founder Common 1주=10표 vs Investor Preferred 1주=1표.
**한국[KR]**: 미상장 벤처회사 개정법(2023.11) 허용 → 조건: 5년 한정, IPO 시 자동 소멸(Sunset Clause).

### 6-3. 의장/대표이사 해임 방지(Anti-Removal)

**표준 조항 ★**:
```
"Founder/CEO 해임은 다음의 경우만:
(1) For Cause (배임, 위법행위),
(2) >50% 공통주 동의,
(3) 또는 자발적 사임"
```

**투자자와 충돌**: Series B+ 부실 경영 시 창업자 교체 요구.

**합의안**: "Board Consent + 투자자 다수결" 조건 제한 + Performance-based KPI 미달 시만 협상.

### 6-4. 반희석 보호(Anti-Dilution) [US]

**Full Ratchet(투자자 유리)**:
```
이전: 1주 = $5
신규: 1주 = $3
→ 이전 투자자 전환 가격: $3 자동 인하
→ 창업자 소유율 심각 희석
```

**Weighted Average(창업자 유리) ★**:
```
공식: NewPrice = OldPrice × 
      (OldShares + NewShares @OldPrice) / (OldShares + NewShares @NewPrice)
→ 공정한 희석 분산 → 표준 창업자 친화
```

**실무**: Series A 이후 AD 필수 → 투자자 요구 → Weighted Average 협상 필수.

## 7. 거버넌스 실패 패턴

### Case 1: 이사회 갈등 → 경영 마비 → 실패

**배경**: 2010년대 중반 미국 인사관리 SaaS (Series B, $15M).
- CEO 2명(공동창업자) vs 투자자 2명(VC펌)

**진행**:
- 분기 1~2: 제품 전략 차이 분명화 (AI 예측 vs 직원 참여도)
- 분기 3: 이사회 투표 2 vs 2 → CEO A 캐스팅 보트로 AI 전략 강행
- 분기 4~6: 투자자 반발 → 보호조항 발동(개발비 승인 거부) → 채용 금지, 마케팅 중단

**결과**: 8개월 내 엔지니어 이탈 40% → 핵심 고객 3개 상실 → Series C 유치 실패 → 18개월 후 수익성 기업으로 구매(2배 평가손실).

**교훈 ★**:
1. **2 vs 2 이사회 최악** → 3인 또는 5인 + 독립이사 중재자 필수
2. **보호조항 과다** → 경영 마비
3. **전략적 의견 차이** → 이사회 아닌 경영진 회의에서 해결
4. **창업자-투자자 신뢰 감소** → 가장 큰 손실

### Case 2: 과도한 보호조항 → 성장 기회 상실

**배경**: 2018년 한국 핀테크 스타트업 선불결제 (Series B).

**보호조항**: 47개 사항에 거부권 (임원 채용·비용 변경·파트너십·기술 스택 모두).

**문제**:
- 월 1: 우수 CTO 영입 기회 → 투자자 회의 3주 지연 → 경쟁사 채용
- 월 3: 대형 은행 파트너십 협상 → 투자자 승인 대기 중 은행-경쟁사 거래 완료
- 월 6: 기술 마이그레이션(레이턴시) → "검증된 기술만" 요구 → 6개월 지연 → 경쟁사 성능 우위

**결과**: 연 성장 목표 200% → 실제 80% → Series C 유치 실패 → Acq-hire 형태 exit (창업자 손실).

**교훈 ★**:
1. **보호조항 많음** → 회사 속도 저하
2. **창업자 협상 실패** → Materiality 임계값 미도입
3. **투자자 과도 개입** → 시드 수준 조항을 Series B서도 유지
4. **시장 기회 상실** → 거버넌스 손실이 투자자 보호보다 크다

## 8. 한국 vs 미국 거버넌스 실무 비교

| 요소 | 미국 [US] | 한국 [KR] | 주의 |
|------|----------|---------|------|
| **표준 용어** | NVCA Term Sheet | 투자계약서(자유) | KR 법제화 낮음 |

| **Drag-Along** | 필수 | 선택 | KR 소수주주 보호 강함 |
| **정보공개** | 분기 보고(SEC 의무) | 관행적(비의무) | KR 규제 약함 |

## 9. 창업자·투자자 체크리스트

### 9-1. 창업자 체크리스트 ★

**시드 라운드 전**:
- [ ] 이사회 3인(창업자 2, 투자자 1) 확인

### 1. 대한상사중재원(KCAB) 중재

**SHA 분쟁 활용**:
- 상설 법정 중재(1966년 설립)
- 2025.3.1부터 개정 국내중재규칙 시행: 전자 중재, 절차 효율화
- 소요기간: **3~6개월**(소송 대비 신속)
- 장점: 기밀성, 최종성(항소 불가), 신속성
- 중재인 선정: 신청금액 5억원↓ 5인 후보, 5억원↑ 10인 후보

**SHA 설계 주의**:
- 중재 대상 범위 명확 정의(의결권 구속 위반 = 중재 O, 주주평등원칙 침해 = 법원 우선)
- 중재상 보전명령(가처분) 사전 검토
- 준거법: 한국 상법 + 국제 상관례 병기 권장

### 2. SHA와 한국 상법 강행규정 충돌

**주주평등원칙(상법 제369조 ①)**: "1주=1의결권" = **강행규정** → 위반 SHA 조항 절대 무효.
**차등의결권(Dual-class)**: KR 불허(미국 Delaware와 핵심 차이).

**의결권 구속약정(Voting Agreement)**:
- 특정 의제 사전 동의서: **유효**
- 이행강제금(liquidated damages): **인정**(대법원 2023.7.13. 선고 2022다224986)
- **한계**: 약정 위반 시 → 주총 결의 자체 유효, 피해자는 **약정 위반 당사자에게만** 손해배상 청구(회사 결의 무효 주장 불가)

**ROFR 위반 구제**:
- 이행 강제(주식 반환): **어려움**(가처분 기각 빈번)
- 손해배상: **가능**(다만 손해액 입증 난함)
- 제3자(양수인) 책임: **불가**(양도 자체는 유효)
- **실무**: SHA 체결 시 ROFR 위반 손해액 **사전에 정량화**(liquidated damages 조항)
