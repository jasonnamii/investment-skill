# L6 — M&A / 매각

## V1. 전략·실행

### M&A 유형

| 유형 | 설명 | 창업자 관점 | 투자자 관점 |
|------|------|-------------|-------------|
| Stock Sale | 주주→주식 매각·회사 완전 자회사화 | 양도세 계산 중요 | 우선주 전환/참가 확인 |
| Asset Sale | 회사 자산·부채 선별 매각 후 해산 | 이중과세 위험 [US] | 워터폴 분배 우선순위 |
| Merger | 법적 합병 | 합병 비율·교부금 협상 | 반대주주 매수청구권 |
| Acqui-hire | 팀·IP 인수·회사 해산 | 고용조건(RSU/보너스) | 투자금 회수 어려움 |

### 타임라인

**[KR]** 매도등록/자문사 → NDA → 2차매칭 → 미팅 → 조건분석 → 딜스트럭처 → MOU → 실사(CDD·FDD) → 본계약 → 클로징 (Term Sheet 체결 시 1~2개월 우선협상 일반)

**[US]** Banker 선정 → Teaser → NDA → CIM → IOI → 경영자 미팅 → LOI → 실사 → SPA/APA (exclusivity 60~90일 표준)

**실사 차이:** [KR] 노무/세무/인허가 3대 | [US] IP/소송/규제 중점

### 딜 스트럭처 핵심

| 요소 | 내용 |
|------|------|
| 대가 | All-cash / Cash+Stock / Cash+Earnout / Stock-for-Stock |
| Earnout | 사후 성과 연동 (매출·EBITDA·사용자 수). 2024~2025: 기간 단축, 실현율 ~21¢/$1 [US] |
| Escrow | 매각가 10~20% 12~24개월 → 진술보장 위반 시 배상 |
| R&W | 매도인 회사 상태 보증. R&W 보험 활용 증가 |
| Non-compete | 창업자 2~3년 동종업 금지. [KR] 기간·범위 제한 판례 |

### 성공 vs 실패 패턴

| | 성공 | 실패 |
|---|---|---|
| 준비 | 12~18개월 전 재무·법무 정비·데이터룸 | "관심 들어왔으니 해보자" → 준비 없이 실사 → 이슈 발견 → 파기 |
| 바이어 | 경쟁입찰로 레버리지 (2~3개 동시) | 단일 바이어 독점 → 주도권 상실 |

### 창업자 체크리스트

- 매각 동기 명확화 (피로·자금·시너지·강제)
- 기존 투자자 동의 필요성 확인 (드래그얼롱·태그얼롱)
- 핵심인력 리텐션 계획
- 세후 수취액 시뮬레이션 (양도세·법인세·원천징수)
- Earnout 시 통제권 유지 방안

## V2. 법률·규제

### [KR] 필수 법규

| 법규 | 장면 |
|------|-----------|
| 상법 §374 | 자산양수도 시 주총 특별결의 필요 |
| 상법 §522 | 합병계약·비율·반대주주 매수청구권 |
| 공정거래법 | 규모별 M&A 신고 의무 |
| 외국인투자촉진법 | 외국인 매수자 신고/허가 |
| 자본시장법 | 상장사 M&A 공개매수·공시 |

### [US] 필수 법규

| 법규 | 장면 |
|------|-----------|
| DGCL | 합병·자산매각 시 이사회+주주 승인 |
| HSR | 기준 초과 시 FTC/DOJ 신고 ($119.5M, 2025) `확인필요` |
| Securities Act | Stock-for-Stock 등록면제 확인 |
| CFIUS | 외국인 인수 시 국가안보 심사 |

### 핵심 계약조항

- **R&W:** 매도인 보증. 위반 시 손해배상 → 에스크로 차감
- **Indemnification:** R&W 위반·세무 이슈 책임 범위·기간·한도
- **MAC/MAE:** 클로징 전 중대 부정적 변화 시 딜 철회
- **Drag-Along:** 다수 주주 매각 결정 시 소수 강제 동반 → [[X2]]·[[L6-드래그얼롱]]
- **Non-Compete/Solicit:** 경쟁·인력빼가기 금지

## V3. 재무·세무

### 밸류에이션

| 방법 | 적용 |
|------|------|
| EV/Revenue | 성장 스타트업 (SaaS 5~15x, 핀테크 3~8x) `2024~2026` |
| EV/EBITDA | 수익화 기업·전통 M&A |
| Precedent Transactions | 유사 딜 기반 |
| DCF | 현금흐름 안정 기업 (스타트업에선 보조적) |

### 세무

| 이슈 | [KR] | [US] |
|------|------|------|
| 주식 양도세 | 중소기업 10%, 대주주 20~25% (2024) `확인필요` | LTCG 0/15/20% + NIIT 3.8%. QSBS §1202 최대 $10M 면제 |
| 자산양수도 | 법인세 + 배당소득세 (이중과세) | 법인 레벨 과세 + 주주 분배 (C-corp) |
| Earnout | 수취 시점 기준 | Installment sale §453 |
| 크로스보더 | 비거주자 양도세 | FIRPTA → [[X3]] |

### 워터폴 (잔여재산 분배)

```
1. 거래비용 (자문수수료·법률비용)
2. 채권자 (대출·CB)
3. 우선주 청산우선권
   - Participating: 우선 분배 + 잔여 지분비례
   - Non-Participating: 우선 분배 OR 전환 후 중택1
4. 보통주 (창업자·임직원·ESOP)
```

> 다운라운드 후 매각 시 participating preferred 있으면 창업자 수취액 극소화 → 시뮬레이션 필수

> **Phase 4 보강** — 교차분석 모순 1.4 해소. L6과 X5 간 Drag-Along 기준값 통일.

| 라운드 | 기준 | 추가조건 | 근거 |
|-------|------|---------|------|
| Seed~A | 50%+1 ~ 2/3 | 이사회 동의 不필요 多 | 투자자 수 소수, 합의 용이 |
| B~C | 2/3 ~ 75% | 이사회 과반수 동의 권고 | 이해관계자 다양화 |
| D+ | 75%+ | 이사회 동의 + Board Consent 필수 | PE/CVC 복잡, 소수주주 리스크 |

**창업자 보호:** 높은 threshold 확보(75%+), Board Consent 조건, Minimum Price Floor 삽입. 초기 협상력 제한 `확인필요`

**[KR] 특수성:** 상법 "주요 재산 처분"(총자산 50%+) 주총 특별결의(2/3) 필수 → SHA Drag-Along이 상법과 충돌 가능 `확인필요`

## 패턴 라이브러리

### L6-P01: 조기 재무·법무 정비 — 성공

- **단계:** L6
- **관점:** V1
- **작동 조건:** 매각 진행 12~18개월 전 재무·법무 정비 착수 + 데이터룸 선제 구축
- **메커니즘:** [준비 태세]→[실사 이슈 사전 해소]→[딜 진행 가속화 + 유리한 협상 조건]
- **신호/지표:** 재무제표 감사 이력, FDD 대응 문서 충실도, 데이터룸 구성 수준
- **대조 패턴:** → L6-P02
- **사례:** "12~18개월 전 재무·법무 정비·데이터룸" (V1 성공 패턴)
- **교차참조:** [[X2-거버넌스]] 이사회 체계 | [[L6-MA]] 타임라인

### L6-P02: 준비 없는 응급 딜 진행 — 실패

- **단계:** L6
- **관점:** V1
- **작동 조건:** "관심 들어왔으니 해보자" 식 즉흥 진행 + 준비 태세 미흡
- **메커니즘:** [준비 부재]→[실사 중 예상 밖 이슈 발견]→[신뢰도 추락 + 딜 파기]
- **신호/지표:** 재무제표 불완정·법무 자료 부재·실사 응답 지연
- **대조 패턴:** → L6-P01
- **사례:** 준비 없이 실사 진행 시 이슈 발견 후 파기되는 사례
- **교차참조:** [[L6-MA]] V1. 전략·실행

### L6-P03: 경쟁입찰 레버리지 — 성공

- **단계:** L6
- **관점:** V1
- **작동 조건:** 동시에 2~3개 바이어로부터 경쟁입찰 상황 유도
- **메커니즘:** [복수 바이어 경쟁]→[판매자 협상력 강화]→[높은 밸류에이션 + 유리한 조건]
- **신호/지표:** 시장 테스트(Teaser/CIM) 반응도, 경쟁 입찰 수 및 조건 다양성

- **교차참조:** [[L6-MA]] V1. 전략·실행 | [[L6-MA]] 타임라인

### L6-P04: 단일 바이어 독점 — 실패
