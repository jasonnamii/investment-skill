## V1. 전략·실행

### 필수지식

#### IPO의 정의 및 생애주기
IPO: 비상장 → 공개시장 상장. 단순 자금조달을 넘어 신용도·거래신뢰·종업원복지 신호. L5는 선제적 성장자금 완료 후 공개시장 진입 포인트.

#### 상장 유형 (한국 & 미국)

**한국:**
- **코스피:** 대형우량(시총 500억원 이상 권장) | 자기자본 300억원, 영업력 3년, 공모주 100만주 | 규제 엄격
- **코스닥:** 창업·중소성장 중심 | 자기자본 30억원, 영업력 1년, 공모주 40만주 | 상장폐지 강화(2026: 150억→2027: 200억→2028: 300억)

**미국:**
- **나스닥:** 기술·성장주 | 순자산 10~25MM USD, 공개주 100만주, $4/주 이상 | 2026: IPO 후 15일(기존 3개월) 지수편입
- **NYSE:** 블루칩·다국적 | 순자산 40MM USD, 공개주 110만주, $4/주 이상 | 보수적평가

#### 상장 준비 타임라인 (24-36개월)
- 월0: 주관사 선정
- 월1-2: DD (재무감사, 법무검토, 경쟁력평가)
- 월3-6: 증권신고서 작성
- 월6-9: 규제심사 + 로드쇼준비
- 월9-12: 로드쇼→수요예측→공모가결정
- 월12: 상장

#### 주관사 선정 기준 (평가순)
| 항목 | 중요도 |
|------|--------|
| 자본시장 경험 (유사산업/규모 IPO 실적 3년) | ★★★★★ |
| 기관 네트워크 (국내/해외 기관투자자) | ★★★★★ |
| 재무가치평가 역량 | ★★★★ |
| 마케팅·로드쇼 성과 | ★★★★ |
| 해외 파이프라인 (듀얼리스팅 경험) | ★★★ |
| 수수료 경쟁력 (3~4.5%) | ★★★ |

**한국 Top:** 삼성증권, KB증권, 미래에셋증권 | **미국 Top:** Goldman Sachs, Morgan Stanley, JP Morgan

#### 공모가 결정 방식
**1단계:** 주관사가 재무제표·경쟁사분석·DCF 기반 공모가 밴드(예: 15,000~18,000원) 설정
**2단계:** 수요예측(Book Building) — 기관투자자만 참여. 과다청약 10배 이상시 공모가 상향신호
**3단계:** 공모가 확정 = 수요예측결과 + 시장여건 + 경영진협상

팁: 로드쇼 성과 80% 결정 → CEO 직접참석 필수. 과도한 공모가 기대 → 락업해제 시 주가폭락 → 스톡옵션가치훼손

#### Direct Listing vs SPAC vs Traditional IPO

| 방식 | 정의 | 장점 | 단점 |
|------|------|------|------|
| **Traditional** | 신주발행 공모 | 자금극대, 투자자확대, 신용도향상 | 규제심사 길음(24~36개월), 수수료 높음(3~4.5%), 공시의무증 |
| **Direct** | 기존주만 상장(신주X) | 더빠름, 수수료저(0.5~2%), 기존주주유동성 | 신자금조달불가, 성장성입증기업만적합 |
| **SPAC** | 상자회사→인수→상장 | 빠른상장(3~6개월단축), DD간소 | 주주이탈위험, 신뢰도낮음(2020-2022붕괴후악화), 수수료높음 |

선택기준: **Traditional**=규모Large·기금多·장기평판중요 / **Direct**=규모Very Large·기금少·속도중요 / **SPAC**=규모Medium·시간제약심함

#### 듀얼리스팅 (해외동시상장)
특징: 자본조달규모극대 + 국제투자자유입 + 글로벌브랜드가치향상 | 규제이중부담(한국거래소+SEC)
기술: ADR 또는 미국자회사독립상장
비용: 국내IPO + 미국S-1 = 10~20억원대 | ROI: 기업규모Large(시총2조원↑)시만긍정적

### 성공·실패 패턴 쌍

#### 패턴 1: 프리-IPO 기업가치평가의 차이

**성공신호:**
- 선수투자자(Seed, Series) 후속라운드 공동투자/상향평가
- 수요예측: 밴드Top 이상, 과다청약율 >10배
- 초일수익률 <100% (과도한기대X)
- 사례: 삼성전자(2023폭등→2024-2026안정), 카카오뱅크

**실패신호:**
- Series 투자자 추가투자거절·인수합병타진
- 수요예측: 밴드하한선 이하, 과다청약 <5배, 기관참여저조
- 초기폭등(300~500%) → 3개월내급락(50~70%)
- 사례: 2021-2022년 K-Tech IPO붕괴

원인: 수요예측의 "시장진정한평가" vs 공모가 괴리 → 초기변동성높음 + 락업해제 후 주주일제매도

#### 패턴 2: 창업자·초기투자자 지분율의 락업구조

**성공(주가안정성高):**
- 창업자지분 > 초기투자자지분 (최소3:2) → 창업자의향강함
- 명확한락업: 최소180일, 차등해제(founder더길게) 공시
- 상장후3개월: 경영진지분증가추세(자사주매입·옵션행사) → 신뢰신호
- 사례: 쿠팡(IPO후창업자적극경영·주가회복)

**실패(주가변동성高):**
- 창업자지분 <30%, 초기투자자 >50% → 경영진인센티브약함
- 불명확한락업: 미공시·30일후즉각해제 → 물량예측불가
- 상장후3개월: PE펀드대규모매도공시 → 투자자실망
- 대조사례: 2022년 상장스타트업 중 PE펀드조기전량매도

원인: 락업 = 신뢰도·투자자의향신호. 초기투자자자금회수욕구 vs 기업성장의trade-off

### 활용노하우

#### 1. 상장타이밍

**시장여건 (2024-2026):**
- 유리: 금리인하사이클, 증시강세, 동종사상장후우호적평가
- 불리: 금리인상, 증시약세, 최근IPO부진

**기업내부여건:**
- 매출성장 가시성: 최근2년 YoY 30% 이상
- 수익성전환: 적자→흑자 또는 명확한경로
- 기업지배구조: 최대주주권한명확, 이사회독립성확보

팁: IPO 1년전부터시장신호모니터링. 6개월전최종결정. 상장연기도 용감한결정(2022년리스팅연기기업중2024년상장후성공다수)

#### 2. IR(Investor Relations) 전략

**수요예측 로드쇼 성패 80%는 경영진프레젠테이션**
- 핵심메시지 3가지: (1)시장기회규모 (2)회사차별화포인트 (3)향후3년매출/영업이익경로
- 기관투자자세분화: Growth investor(매출성장률강조) / Value investor(NAV·EPS강조) / Dividend investor(배당정책명확화)
- 로드쇼일정: 국내2주→미국1주(듀얼리스팅)→유럽5일

팁: CEO·CFO 직접참석 → 신뢰↑. 바이사이드만남시 재무Q&A철저준비

#### 3. 공모가극대화전략

**밴드상향협상:**
- 수요예측사전예고: 기관에 비공식신호 → 높은공모가기대조성
- 선수투자자동향활용: "Series D예정" → 공모가상향심리자극
- 시장신호: 상장1개월전직전분기실적좋으면공시타이밍조절

**공모규모조정:**
- 과도하게큰공모: 주가하락 + 기관수요부족리스크
- 최적범위: 시총의 15~25% (유동성+수요 모두만족)

팁: 공모가높을수록향후주가상승압력약해짐 → 적정수준공모가가장기투자자수익률+기업자금조달동시달성

### 계약서핵심조항

#### 1. 주관사계약 (Underwriting Agreement)

**주요조항:**
- **공모규모·공모가:** 예비합의→최종합의시점
- **주관사배정:** 대표주관(Lead) vs 부주관 역할분담
- **수수료:** 기본(1~1.5%) + 판매(1~2%) + 인수(0.5~1%) = 총3~4.5%
- **Lock-up:** 상장일180일(또는240일) 주요주주매도금지
- **Green Shoe Option:** 공모량15% 범위내 추가공모권(상장후30일이내)
- **표현및보증:** 회사진술·보증책임
- **손해배상:** IPO분쟁시책임할당

팁: 공모규모크면(500억원↑) 3.5% 이하협상가능. 락업240일은주가안정성신호. 그린슈상장후주가강세예상시최대활용

#### 2. 주요투자자Exit조건 (SPA수정)

**ROFR (Right of First Refusal):** 창업자매도시기존투자자먼저기회제공 → IPO상황에서일반적해제(유동성확보)

**Lock-up차등구조:**
- Founder: 180일(또는240일)
- Series투자자: 180일
- 직원(옵션): 90일또는180일

**할당비율:** 공모물량중 기관90% / 개인10% (한국스탠다드)

#### 3. 락업(Lock-up) 핵심이해

**의미:** 상장초기기존주주의주식매도제한 → 주가안정성확보 = IPO투자자신뢰척도

**해제시점:**
- 1차: 상장일+180일(표준)
- 2차: 상장일+240일(강한신뢰신호)
- 조기해제: 주가안정성판단시주관사협의→단축가능

**해제후주가영향:**
- 대규모매도예상 → 주가약세
- 예외: 실적우호적 또는 경영진추가매입신호시 → 주가상승

#### 4. 그린슈옵션 (Greenshoe Option)

**정의:** 공모가이상거래시추가주식(공모량15%범위)을공모가로발행가능한권리

**메커니즘:** 상장후주가 > 공모가 → 주관사그린슈행사 → 추가공모

**비용:** 그린슈행사시 → 창업자/주주지분희석(0.5~3%)

팁: 상장후주가 ≥ 공모가의120%면그린슈행사확률높음 → 추가희석계획필수

### 밸류에이션

#### 공모가결정의재무방법론

**1. PER(Price-Earnings Ratio) 방식**

```
공모가 = 예상1주당순이익(EPS) × 목표PER

예시:
- 순이익 100억원 / 공모주 5,000만주 = EPS 2,000원
- 동종사평균PER 15배 + 성장프리미엄 = 목표PER 18배
- 공모가 = 2,000원 × 18 = 36,000원
```

해석: PER높음=성장기대높음(성장주) / PER낮음=저평가(가치주). IPO기업은동종사평균대비10~30% 프리미엄가능

팁: 1년후추정EPS보수적산정(예: 20%성장) → 공모가정당성제시

**2. PSR(Price-to-Sales Ratio) 방식**

```
공모가 = 예상1주당매출액 × 목표PSR

예시:
- 매출 500억원 / 공모주 5,000만주 = 1주당매출 10,000원
- 동종사평균PSR 2배 + 마진프리미엄 = 목표PSR 2.5배
- 공모가 = 10,000원 × 2.5 = 25,000원
```

사용시기: 적자기업(EPS음수) 예: 배달앱·이커머스초기

**3. DCF(Discounted Cash Flow) 방식**

```
내재가치 = Σ(미래자유현금흐름의현재가치) + 터미널가치
공모가 = 내재가치 / 공모후전체주식수
```

프로세스:
- 5년자유현금흐름(FCF) 전망: 매출성장률·영업이익률·자본지출
- 할인율(WACC): 회사자본비용(일반8~12%)
- 터미널가치: 5년이후영구적성장(일반2~3%)

예시계산:
```
Year 1-5 FCF = 20억, 35억, 50억, 60억, 70억원
WACC = 10%, 터미널성장률 = 2%

현재가치합계(Year1-5) = 169.12억원
터미널가치 = 442.25억원
내재가치 = 611.37억원
공모주 5,000만주 → 공모가 = 12,227원
```

해석: DCF는 "가장정교하나가정에민감" (WACC·성장률1% 변화 → 공모가10~15% 변동)

#### IPO디스카운트 vs 프리미엄

**디스카운트 (할인상장):**
- 공모가 < 마지막사모라운드가격
- 원인: 시장악화 / 기업실적부진 / 과도한공모규모
- 신호: 부정적(기업가치하락)
- 2024-2026사례: 시장조정중신규IPO일부

**프리미엄 (상향상장):**
- 공모가 > 마지막사모라운드가격
- 원인: 시장강세 / 매출이익우호성장 / 높은투자자수요
- 신호: 긍정적(성장기대)
- 표준: 20~40%범위건전 (>50%=버블신호)

팁: Series D $50M(시총$500M) → 공모가$550~600M범위프리미엄10~20% 추천. 과도시상장후주가부진 → 스톡옵션가치훼손

### 세무이슈

#### [KR] 양도소득세및대주주세금

**기본과세구조:**

```
양도소득 = 매각가 - 취득가
양도소득세 = 양도소득 × 20~40% (보유기간·금액차등)
```

| 보유기간 | 세율 |
|---------|------|
| 1년미만 | 40% |
| 1~5년 | 30% |
| 5년이상 | 20% |

**대주주특별세:** 최대주주공시주식양도시추가제한 → 기업지배력남용방지

**예시 (창업자):**

```
공모가 30,000원, 지분 1,000만주(20%), 1년후주가 50,000원

시나리오1: 1년후전량매도
- 매각금액 = 500억원
- 취득가 = 1,000만원(무상·저가취득가정)
- 양도소득 ≈ 500억원
- 양도소득세(30%) = 150억원
- 순수익 = 350억원

시나리오2: 5년후매도
- 양도소득세(20%) = 100억원
- 순수익 = 400억원
```

**절세전략:**
1. 분할매도: 연간매도규모제한 → 누진세율경감
2. 신고이연: 5년이상보유후매도 → 20% 적용
3. 기부활용: 일부주식재단·대학기부 → 양도소득공제

#### [KR] 스톡옵션행사및세무

**2단계과세구조:**

```
Step1: 행사시점
→ 행사차익 = (행사일주가 - 행사가) × 행사주수
→ 근로소득세(재직) 또는 기타소득세(퇴사후)

Step2: 주식매도시점
→ 양도소득 = (매각가 - 행사일주가) × 행사주수
→ 양도소득세
```

**벤처기업스톡옵션조세특례 (2024-2026현행):**

| 특례 | 내용 |
|------|------|
| **비과세** | 행사차익 1인당연간2억원, 누적5억원까지 |
| **과세이연** | 행사시소득세미납 → 매도시양도소득세일괄 |
| **분할납부** | 비과세제외부분의소득세를5년분할 |

**예시 (벤처기업직원):**

```
행사가 10,000원, 10,000주, 행사일주가 30,000원
행사차익 = 2억원

비과세특례적용 (연간2억한도):
→ 행사차익2억원전액비과세 (근로소득세0원)
→ 5년누적한도5억원이므로향후3억원추가가능

3년후주식매도시 (매가50,000원):
- 양도소득 = 2억원
- 양도소득세(30%) = 6,000만원
- 순수익 = 1.4억원
```

**IPO타이밍전략:**
- IPO직전(1~3개월): 행사 → 행사일주가낮음 → 행사차익최소화 → 세금절감
- IPO후6개월(락업근처): 행사 → 행사일주가높음 → 행사차익크지만 "과세이연특례" 활용가능
- 추천: 반반분할(IPO전+후)

*확인필요*: 2026년스톡옵션세제개정논의중 (행사차익한도상향 또는비과세율확대예상)

### 1. 거버넌스전환타임라인

**[US] NYSE/NASDAQ 독립이사요건:**
- IPO시점: 최소1명독립이사(Audit, Compensation, Nominating 각위원회)
- 90일이내: 각위원회과반수(majority) 독립이사구성
- 1년이내: 이사회전체50%↑ 독립이사(controlled company제외)

**[KR] 상법/KRX요건:**
- 상법제542조의8: 이사총수의 **1/3이상** 사외이사선임
- 감사위원회필수구성(사외이사포함)

**VC이사이탈패턴 (모순1.5 해소):**
- 락업기간: 180일~240일
- VC주식보유기간: IPO후평균 **3년** (장기보유경향)
- 락업만료시: VC-backed기업이 non-VC기업대비 **더큰주가하락** 경험
- **거버넌스공백:** VC이사사임시시장음의신호반응 → 후임독립이사사전확보필수
- 실질전환: 투자자이사 "형식적잔류" → "사임+매도"로전환되는시점이거버넌스리스크최대

### 2. Post-IPO실적함정

**IPO Window Dressing후실적갭:**
- IPO전가장공격적수익관리(상위25%)기업은, IPO1년후 **15~25% 낮은수익률** 기록
- 원인: Pre-IPO accrual manipulation(단기현금유입극대화, 지속성없는비용절감)
- 시장반응: 조작실적은 **1~3년에걸쳐점진적으로드러남**

**구조적실적악화원인:**

| 요인 | 메커니즘 |
|------|---------|
| 핵심인력이탈 | 락업만료후초기직원·임원의liquidity event → 퇴사(CTO·CFO급) |
| 경영진인센티브약화 | 옵션행사가속화 → post-IPO동기저하 |
| 분기공시압박 | 단기실적부스트목표 → R&D·시장확장지연 |
| 의사결정속도저하 | 이사회위원회구조화 → 비공식결정불가 → CEO권한실질축소 |

**분기실적미달페널티:** 목표미달 = 통상다음날 **-5~10% 주가하락**

### 3. 창업자보호전략 (모순1.3 해소)

> **모순1.3 교정:** Series B+ 이후창업자는이사회다수결을상실한상태에서IPO진행. 공모가협상주도권제한적 → 아래보호장치가 **사전에** 확보된경우에만실질적영향력행사가능

**Dual-Class Voting (Class A/B) [US]:**
- Class A: 1주=1표(일반투자자)
- Class B: 1주=10표↑(창업자전용)
- 2017~2019: 약30% IPO기업이채택
- Sunset clause필수: 지정기간후 Class B → Class A자동전환
- **[KR] 한국은차등의결권불허** (상법상1주1의결권강행규정)

**Founder-Friendly이사회설계:**
- Reserved Board Seats: 창업자지정이사석(이사회결의로제거불가)
- Tie-breaking Vote: 동수시창업자최종투표권
- Consent Right: M&A·자산매각·핵심인사에창업자동의필수

**현실제약:** 창업자보호강할수록기관투자자저항증가 → "founder-friendly but still investable" 타협점확보가핵심

### L5-P01: 프리-IPO 기업가치 평가의 정확성 — 성공

- **관점:** V1, V3

- **메커니즘:** [정확한 밸류에이션 + 투자자 신뢰] → [높은 기관수요] → [적정 공모가 + 장기 주가 안정]

- **사례:** 삼성전자(2023 상장공모가 대비 지표 안정화), 카카오뱅크(수요예측 기관수요 강세)

### L5-P02: 프리-IPO 기업가치 평가의 오류 — 실패
- **단계:** L5
- **관점:** V1, V3
- **작동 조건:**
  - Series 투자자 추가투자 거절 또는 기존 지분 감소 신호
  - 수요예측 밴드하한 이하, 과다청약 <5배, 기관참여 저조
  - 초기폭등(300~500%) 직후 3개월 내 급락(50~70%)
- **메커니즘:** [과도한 공모가 기대] → [시장 진정한 평가 괴리] → [초기변동성 + 락업해제 후 대량매도]
- **신호/지표:**
  - Series 라운드 이후 기관 투자자 네트워크 위축
  - IPO 신고 후 시장 악화로 공모가 밴드 하향 조정 다회 반복
  - 초일 거래 후 5거래일 내 하락폭 >20%
- **대조 패턴:** → L5-P01 (프리-IPO 기업가치 평가 정확성)
- **사례:** 2021~2022년 K-Tech IPO 붕괴 사례군 (초기 과도 평가 후 상장 1년 내 주가 50% 이상 하락)
- **교차참조:** X5-투자자창업자관계(§투자자 신뢰 상실), VX-밸류에이션통합(§공모가 밴드 결정 오류)

### L5-P03: 창업자·초기투자자 지분율 락업 구조의 효과 — 성공
- **단계:** L5
- **관점:** V2, V1
- **작동 조건:**
  - 창업자 지분 > 초기투자자 지분 (최소 3:2)
  - 명확한 차등 락업: Founder 240일, Series투자자 180일 공시
  - 상장 후 3개월 내 경영진 자사주 매입 또는 옵션 행사 공시
- **메커니즘:** [강한 창업자 지분 + 차등 락업] → [시장 신뢰 신호] → [주가 안정성 + VC 이사 조기 이탈 회피]
- **신호/지표:**
  - 상장 후 3개월 내 경영진 지분 증가 트렌드
  - 락업 해제 시점 근처 매도 물량 예측 정확
  - 상장 후 1년 내 주가 변동성(Volatility) 동종사 평균 이하
- **대조 패턴:** → L5-P04 (창업자 지분 약화로 인한 락업 리스크)
- **사례:** 쿠팡(IPO 후 창업자 적극 경영 + 주가 회복 추세)
- **교차참조:** X1-지분설계(§founder-friendly 구조), X5-투자자창업자관계(§락업 신뢰도)

### L5-P04: 창업자·초기투자자 지분율 락업 구조의 실패 — 실패
- **단계:** L5
- **관점:** V2, V1
- **작동 조건:**
  - 창업자 지분 <30%, 초기투자자 >50% (경영진 인센티브 약함)
  - 불명확한 락업: 미공시 또는 30일 후 즉각 해제
  - 상장 후 3개월 내 PE 펀드 대규모 매도 공시
- **메커니즘:** [약한 창업자 지분 + 불명확한 락업] → [투자자 신뢰 상실] → [락업 해제 직후 대량 매도 + 주가 급락]
- **신호/지표:**
  - IPO 신고 당시 창업자 지분 공시 후 기관 투자자 부정적 피드백
  - 락업 해제 1주일 전 기관 투자자 감소 추세
  - 상장 후 3개월 내 주요 VC/PE 펀드 전량 매도 공시
- **대조 패턴:** → L5-P03 (창업자 강한 지분 구조의 신뢰 신호)
- **사례:** 2022년 상장 스타트업 중 PE 펀드 조기 전량 매도 사례군 (3개월 내 주가 40~60% 하락)
- **교차참조:** X1-지분설계(§기관투자자 선호 구조), L7-세컨더리(§Lock-up 해제 후 매도 패턴)

### L5-P05: 상장 타이밍 선택 — 성공 (시장 우호)

- **메커니즘:** [시장 여건 호황 + 기업 성장 신호] → [높은 기관 수요] → [공모가 상향 + 상장 후 주가 강세]

- **대조 패턴:** → L5-P06 (부실한 상장 타이밍)
- **사례:** 2024년 한국 금리 인하 사이클 진입 후 IPO 기업들 평균 초과수익률 양수
- **교차참조:** 교차분석(§빈자리 3.2: 상장 타이밍 전략), X5-투자자창업자관계(§시장 사이클 활용)
