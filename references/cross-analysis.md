## 4. L×X 교차 인사이트 (3건)

### 4.1 Exit 신뢰도 역설 (L5-L7 × X2 × X5)

세 exit 경로를 X2(거버넌스)+X5(관계)와 교차하면 **"exit = 투자자 우선 회수의 제도화"** 구조 노출:

| 경로 | 형식적 의미 | 구조적 현실 |
|------|-----------|----------|
| **L5 IPO** | 성공의 정점 | X2: 독립이사 100% → 창업자 통제권 미상장 Series C 때보다 약함 |
| **L6 M&A** | 최종 매각 | X5: Drag-Along + 어닝아웃 = 창업자를 "인수 후 고용인"으로 전환 |
| **L7 세컨더리** | 유동성 제공 | X5: 펀드 만기 7-10년 회수 압력 → 창업자 보유 금지 구조 |

**핵심:** 형식상 선택지이나 구조적으로 3경로 모두 투자자 우선 회수로 수렴. 창업자의 실질 선택권은 dual-class(L5)나 높은 Drag-Along threshold(L6, X5)를 **사전에** 확보한 경우에만 존재.

### 4.2 예측 실패의 증폭 회로 (X4 × L6 × X5)

```
X4: Series A Crunch (PMF 미확정) → 펀딩 실패
↓
L6: 약한 입장에서의 M&A 협상 → Drag-Along 51% 적용
↓
X5: 투자자-창업자 관계 악화 → 추가 투자 거절
↓
[자기실현 예언: 6개월 내 강제매각]
```

세 축 미결합 시 각각이 "객관적 구조"처럼 보이나, 교차하면 **초기 실패 신호 1회 → 6개월 후 강제매각**의 가속 경로 드러남. 투자자의 "합리적 판단"(후속 거절)이 사실상 회사를 죽이는 메커니즘.

**대응:** X4에 "예측 실패 조기 신호(9개월 전) + 대응 전략" 추가. 증폭 회로 차단 시점은 "런웨이 6개월 전"이 아니라 "12개월 전".

### 4.3 세무 리스크의 숨은 주체 변화 (X3 × L5-L7)

| Exit 경로 | 세무 책임 주체 | 핵심 이슈 | 리스크 구조 |
|----------|-------------|---------|-----------|
| **L5 IPO** | 회사(Korea Inc) | Transfer pricing + CFC 귀속과세 | CFO·회계팀 담당, 회사 내부 |
| **L6 M&A** | 창업자 개인 | 양도소득세 [KR] 20~25% + QSBS [US] | 세후 수익 L6 워터폴과 별개 |
| **L7 세컨더리** | 매도 주주(분산) | 비상장주식 양도소득세 | X3 Transfer Pricing과 독립 |

**핵심:** 창업자는 L5선택 시 회사의 CFC 문제 간접 부담, L6선택 시 양도세 직접 부담. Exit 경로 선택 = 세무 리스크 부담 주체 결정. 개별 문서에 미명시.

## 부록 A. Exit 경로 선택 매트릭스

### A.1 선행 조건 (진입 자격)

| 조건 | L5 IPO | L6 M&A | L7 세컨더리 |
|------|--------|--------|-----------|
| **연매출** | $50M+ (KOSDAQ $30M+) | 제한 없음 | 제한 없음 |
| **수익성** | 영업이익 흑자 또는 경로 명확 | 불필요(기술가치도 가능) | 불필요 |
| **성장률** | YoY 30%+ 안정적 | 둔화 시에도 가능 | 무관 |
| **시장 환경** | IPO 윈도우 필수 | 바이어 수요 존재 | 세컨더리 펀드 유동성 |
| **거버넌스** | 독립이사 50%+(KR)/100%(US), SOX | 이사회 동의 + Drag-Along 충족 | ROFR·동의권만 |
| **준비기간** | 24~36개월 | 12~18개월 | 3~6개월 |
| **최소 단계** | Series C+ | 제한 없음 | Series B+ |

### A.2 의사결정 트리 (3단계 필터)

**1단계: 런웨이 필터**
- < 6개월 → L6(M&A) 또는 청산 (선택지 축소, X4 증폭회로)
- 6~18개월 → L7(세컨더리) + 추가 펀딩 병행
- > 18개월 → 3개 경로 모두 열림 → 2단계

**2단계: 시장·투자자 시그널 필터**
- IPO 윈도우 개방 + 투자자 exit 없음 → L5(IPO)
- 인바운드 M&A + 전략적 시너지 명확 → L6(M&A)
- 펀드 만기 임박(2년 이내) + LP 회수 압력 → L7(세컨더리)
- 복수 시그널 → 3단계

**3단계: 창업자 목표 필터**
- 통제권 유지: L5(dual-class) > L7 > L6
- 현금 회수 극대: L5(고밸류) > L6(프리미엄 바이어) > L7
- 시간 효율: L7(3~6개월) > L6(12~18) > L5(24~36)
- 팀 보호: L7(회사 존속) > L5(성장) > L6(조건 협상)

### A.3 경로별 핵심 리스크

| 리스크 | L5 IPO | L6 M&A | L7 세컨더리 |
|--------|--------|--------|-----------|
| **통제권 상실** | 높음 — 독립이사 다수+공시 의무 | 최고 — 어닝아웃 기간 고용인 전환 | 낮음 — 회사 존속, 의사결정 유지 |
| **밸류에이션 리스크** | 시장 변동 노출(공모가 협상) | 바이어 1~2명 의존 | 할인율 변동(3~46%) |
| **시간 리스크** | 24~36개월 + 락업 180~240일 | 클로징 지연·딜 파기 | 비교적 낮음 |
| **세무 부담** | 회사(CFC·TP) | 창업자 개인(양도세) | 매도 주주 분산 |
| **관계 리스크** | 투자자 이사 이탈 후 거버넌스 공백 (모순 1.5) | Drag-Along 갈등 (모순 1.4) | ROFR 미이행 분쟁 |

### A.4 병행 전략 패턴

| 패턴 | 설명 | 적용 시점 |
|------|------|----------|
| **L7 → L5** | 세컨더리 유동성 제공 → 안정 후 IPO | Series C+, IPO 2년+ 전 |
| **L6 // L5** | M&A 탐색과 IPO 준비 동시 진행, 더 유리한 쪽 선택 | Growth, 바이어 인바운드 시 |
| **L7 → L6** | 세컨더리로 시간 확보 → 전략적 매각 | 런웨이 부족이나 즉시 매각 불리할 때 |
| **L5 실패 → L6** | IPO 윈도우 닫힘/실적 미달 → M&A 전환 | IPO 준비 중 시장 악화 시 |
